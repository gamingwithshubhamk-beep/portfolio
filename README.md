## 📂 Repository Contents

* `index.html` — The main glassmorphic single-page portfolio website. Includes custom tilted 3D cards, a WebGL particle node connection background, and the interactive J.A.R.V.I.S. Core SSH Simulator.
* `resume.md` — Complete developer resume in Markdown.
* `resume.pdf` — Compiled professional PDF resume.
* `build_pdf.py` — Python compilation script using ReportLab to generate `resume.pdf` with balanced layouts.
* `shubham.jpg` — Portfolio avatar picture.
* `interview_prep_notes.md` / `interview_prep_notes.pdf` / `build_notes_pdf.py` — Systems Engineering and CS core prep resources.

---

## 🛠️ How to Use Locally

### 1. View the Portfolio
Simply open `index.html` in any web browser to explore the portfolio and play with the terminal simulator locally.

### 2. Compile the PDF Resume
If you update `build_pdf.py` and want to compile a new copy of `resume.pdf`, run:
```bash
pip install reportlab
python build_pdf.py
```
